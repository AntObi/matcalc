Initialize a NEBCalc from end images.

Parameters:

  • start_struct (Structure) – The starting image as a pymatgen Structure.

  • end_struct (Structure) – The ending image as a pymatgen Structure.

  • calculator (str | Calculator) – ASE Calculator to use. Defaults to M3GNet-MP-2021.2.8-DIRECT-PES.

  • n_images (int) – The number of intermediate image structures to create.

  • interpolate_lattices (bool) – Whether to interpolate the lattices when creating NEB path with Structure.interpolate() in pymatgen. Defaults to False.

  • autosort_tol (float) – A distance tolerance in angstrom in which to automatically sort end_struct to match to the closest points in start_struct. This argument is required for Structure.interpolate() in pymatgen. Defaults to 0.5.
